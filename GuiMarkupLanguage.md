## What is GML ##
GML is a GUI language for SerenityOS. It is used to make UIs for apps.




## GUI::Widget ##
GUI::Widget is used as a canvas for other items.
<br>
For Example:
<br>
<br>
Open GML Playground and create a Widget by writing this:

    `@GUI::Widget {
      @GUI::Button {
        text:"Widgets!"
      }
    }`
